# <a name="sku-optimization-solution-guide"></a>SKU 优化解决方案指南

对于零售商，当供不应求时，缺货就会发生。 这是个坏消息，但对于消费者品牌而言甚至更糟：购物者在面对空货架时可能会尝试购买竞争对手的产品，进而导致品牌损失。 解决办法就是 SKU 分类优化，它可确定产品/服务的最佳广度。  

本解决方案指南逐步介绍了方案，以及新分析引擎是如何提供 SKU 分类见解的。
